# How to IntelliJ

- `poetry install`
- Run `./venv.sh`
- Open folder with Idea IntelliJ
- Setup SDK for the module
  - Add Python SDK
  - Virtualenv
  - Existing Enviroment
  - Add `./venv/bin/python` executable
- Select the new created env/sdk.
- Go to project, set the same sdk.
- Save.
- Profit.
